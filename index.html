TraveloZity

A site to plan your travels and look up live flight and hotel availability. This site is integrated with two Amadeus APIs, the Flight Offers and Hotel Offers API. A user is able to search live hotel and flight data to add it to their plan while on the site. This client for this site was made using JavaScript, React, HTML, Axios, and CSS/SASS, while the API was built using Django and Python.

NOTE: The Amadeus APIs used are sandbox APIs, and therefore have a monthly limit for calls. Also, due to the limitations of the hotel API, this site is configured to show booked hotels as well as free ones. If a hotel's price says 'N/A', that mean's it's already booked. Free hotels will show the actual price.

Homepage Depot

A store selling magical goods for programmers. This project was done in a group of three. This site integrates with Stripe API to allow users to make 'purchases'. The client was made with React, JavaScript, Axios, HTML and CSS, while the API was made with Express.js, Lodash, Mongoose, MongoDB and JavaScript.

NOTE: No actual payments can be made using this app. It uses Stripe's test mode, a card number of '4242424242424242', a security code of '424' and an expiration date of 'April 2022' can be used to test it.
